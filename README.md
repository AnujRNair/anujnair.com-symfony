AnujNair.com-symfony readme
===========================

## Setup

* `composer install`
* `npm install`

## Development

* `php app/console assets:install --symlink --relative`
* `php app/console server:run`
* `php app/console assetic:watch`

## Production

* `./tools/build.sh`
